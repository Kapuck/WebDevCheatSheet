Basic HTML Tags

HTML Base Anatomy

Tag Name Description
<!DOCTYPE html> doctype It is required in the preamble. It tells the browser to expect HTML
<html> html The html element, also known as the root element, surrounds the entire contents of the page except the doctype
<head> head The head element contains all of the non-displayable content you want to include in the page, including links to css files, keywords, character set declarations, etc.
<title> title The title element contains the title of the page as it would be displayed in the browser tab. It is also what's displayed when the page is bookmarked or saved.
<body> body The body element contains all of the data you wish to display to your audience including text, photos, videos, audio, and other media.

Multimedia Tags

Tag Name Description
<img> image The image element is a self-closing tag that can display an image using the 'src' attribute, which contains a path to the image file you wish to display. This element can also display alternate text with the 'alt' attribute for visually impaired users or when the image does not load properly.

Text Markup Tags

Tag Name Description
<h1>, <h2>, <h3>, <h4>, <h5>, <h6> heading The headings element signifies headings and subheadings for content, similar to chapter titles. <h1> is the largest heading down to <h6>, the smallest.
<p> paragraph The paragraph element contains paragraphs of text.

List tags

Tag Name Description
<ul> unordered list Unordered lists display with each <li> element featuring a bullet in front of the text. They are for lists where the order does not matter
<ol> ordered list Ordered lists display with each <li> element featuring a number in front of the text. The order of these lists is important.
<li> list item A list item element contains a distinct item within the list.
